My Projects

🎹 Piano Pitches 🎹

My final group project at Dev Academy Aotearoa. We took the beautiful voices of our cohort singing 'Whai' (our cohort name), and added them to our keyboard. There is also the ability for other users to add their own unique voice to our playground. We were able to incorporate a helper bot named Mini, who had tips and tricks for using our site. Tech used: TypeScript, JavaScript, CSS, Cloudinary, React. This has been optimized for Desktop only, as this project is a work-in-progress!

📚 Renees Book Club 📚

My first personal project, started while I was at Dev Academy Aotearoa. I love reading and wanted to create a website where you can store books you've read, you want to read, get recommendations and search for books to read. During this project I fell in love with beautiful, responsive front-end development, and incorporated stunning colour gradients. Tech used: TypeScript, JavaScript, CSS, HTML, React. This has been optimized for Desktop only, as this project is a work-in-progress!
